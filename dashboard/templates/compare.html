📋 Evaluation Methodology

📐 Metrics Explained

MetricFormulaInterpretation
MAEmean|y−ŷ|Average absolute error
RMSE√mean(y−ŷ)²Penalises large errors
MAPEmean|y−ŷ|/y ×100% error (scale-free)
SMAPESymmetric MAPERobust to near-zero
R²1−SS_res/SS_totVariance explained
Adj R²1−(1−R²)(n−1)/(n−k−1)Penalises extra features

🔬 Evaluation Setup

  • Temporal train/test split (no data leakage)
  • Test set = last 20% of time-ordered data
  • XGBoost: TimeSeriesSplit 5-fold CV
  • LSTM: validation_split=0.1 on training data
  • ARIMA: ADF stationarity test before fit
  • All models evaluated on same test window
